# Overview

Instead of using Plugins or APIs to get your step data. This app uses your devices accelerometer to determine the steps taken.

## Usage

### METHOD 1: Using the script directly 

1)Download the Script.gd file

2)Add the file to your godot project

3)Attach the file to a Node2D in Godot

\

### METHOD 2: Downloading the Godot project itself

1)Download the folder "main-app"

2)Click "import" in Godot

3)Select your folder and import it

----------

## Pros and Cons of this Project

### PROs:

1)Does not require any user permissions

2)Does not utilise any APIs or Plugins

3)Very simple to use

4)Easy to modify variables

### CONs:

1)Inaccuracy of ~6% (For every 100 steps taken there will be 10 steps added/removed on average)

2)Requires your device to have an accelerometer
